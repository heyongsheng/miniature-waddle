# 单图预览

单图预览用到的配置如下

字段 类型 默认值 备注
url String 无 预览的图片地址，多图预览时省略

# src

如果您需要预览单图，仅需将图片地址作为参数 url 传入即可。

this.$hevueImgPreview(url)
// 当您有其他配置项需要修改时，也可这样写
this.$hevueImgPreview({
  url: url,
  keyboard: true
})
